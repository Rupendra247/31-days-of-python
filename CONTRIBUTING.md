# Contributing to 31 Days of Python

Thank you for your interest in helping improve this learning resource! 

## How to Contribute
1. **Report Issues:** If you find a bug in the code or an error in the explanations, please open an Issue.
2. **Submit Improvements:** If you have a more "Pythonic" way to solve a daily challenge:
   - Fork the repository.
   - Create a new branch.
   - Make your changes.
   - Open a Pull Request.

## Coding Standards
- Please maintain the current structure (Day-by-day files).
- Ensure all new code includes comments explaining the logic.
- Keep the documentation clear and beginner-friendly.

*Happy coding!*
